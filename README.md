# carton

`carton` 是一个围绕纸箱封装方式展开的项目介绍。

在这个设定里，纸箱不是普通容器，而是一套打包系统，也是一种快速打包和组装技术。

它的核心价值不在于材料本身，而在于三个动作：

- 快速收拢内容
- 快速形成外部结构
- 快速进入可搬运、可堆叠、可交付状态

从这个角度看，纸箱并不只是包装材料，更像一种面向现实问题的轻量级组装方案。它把原本松散、裸露、难以移动的内容，迅速整理成一个可以运输、陈列、储存和再次开启的整体。

## 系统介绍

`carton` 是一套面向封装、打包和组装场景的纸箱系统。它关注的是如何用直接、稳定、低成本的方式，把分散内容快速整理成可处理的整体。它关心的问题包括：

- 什么结构可以在最短时间内完成打包
- 什么折叠方式最适合快速组装
- 什么尺寸最适合搬运、堆叠和重复利用
- 什么外观最容易传达“已经被整理好”的状态

## 设计原则

- 快速：优先完成打包和成型
- 轻量：用尽量简单的结构解决问题
- 易组装：不依赖复杂步骤
- 可封装：把散乱内容迅速收束成整体
- 可展开：封起来方便运输，打开后方便使用
